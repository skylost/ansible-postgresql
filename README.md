# Ansible role Postgresql

Ansible role for Postgresql with pgdg repository

## Requirements

This playbook requires Ansible 2.0.0.1 or higher.

## Features

* Install PostgreSQL

## Running playbook

```bash
ansible-playbook -i hosts posqtgresql.yml
```
